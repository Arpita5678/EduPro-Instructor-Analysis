 ## EduPro Instructor Analysis (Power BI Project)

## 📊 Overview
This project analyzes instructor performance and course ratings using Power BI.

## 📁 Dataset
- Courses
- Teachers
- Transactions

## 📌 Key Insights
- Average Teacher Rating: 4.42
- Average Course Rating: 4.46
- Average Experience: 9.6 years
- Top courses and categories identified

## 📈 Visualizations
- Bar charts (Course & Category analysis)
- Scatter plot (Experience vs Rating)
- KPI Cards

## 🛠 Tools Used
- Power BI
- Excel
- Project report (PDF/Word)
